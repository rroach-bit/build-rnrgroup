# RNR Build — Technological Construction Consultancy

A modern, responsive website for RNR Build, a technological building construction consultancy company based in the Cayman Islands.

## Pages

- **`index.html`** — Main website (single-page, fully responsive)
- **`brand-identity.html`** — Brand identity guide (colors, typography, logo usage, collateral)

## Brand Identity

- **Colors:** Navy (#1B2D4F) + Teal (#00A5A8) dual-anchor palette
- **Fonts:** Outfit (headings), DM Sans (body), Space Mono (labels/data)
- **Tagline:** Building Smarter. Building Better.

## Tech Stack

- Pure HTML/CSS/JS — no build step required
- Google Fonts (Outfit, DM Sans, Space Mono)
- Scroll-triggered animations via Intersection Observer
- Fully responsive (mobile, tablet, desktop)

## Deployment

Deploy as a static site on Vercel, Netlify, or any web host.
